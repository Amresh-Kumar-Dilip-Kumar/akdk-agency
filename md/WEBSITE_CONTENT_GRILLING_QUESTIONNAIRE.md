# AKDK Website Grilling Questionnaire

Purpose: Collect raw, specific business truth so website copy is sharp, credible, and conversion-focused.

Date: March 18, 2026
Prepared for: AKDK Digital content discovery

---

## How to Answer (Mandatory Rules)

1. Do not answer with generic words like quality, innovative, best, affordable, trusted, end-to-end.
2. Every claim must include proof, example, metric, or specific story.
3. If you do not know, write Unknown and assign an owner + deadline.
4. If a number is used, include source and date.
5. Use real names, real tools, real timelines, real constraints.
6. If a question does not apply, write Not Applicable and why.

Minimum quality standard for each answer:
- Specific
- Verifiable
- Customer-relevant
- Decision-useful
